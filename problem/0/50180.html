Task Description

Given two nn by nn square matrices AA and BB, write a program to compute the matrix C=ABC=AB.

Every row of AA, BB, and CC will be stored sequentially in one file. All file names for a matrix have the same prefix, and end with its row index start from 11 to nn. For example, if the prefix for AA is sample1A, then the first row of AA will be stored in a file sample1A1. All elements in AA and BB are int32_t, and the elements of CC are int64_t. You should compute CC and store it in files.

Input Format (stdin)

The first line is the size of the matrices, nn. The second to the fourth lines are the filename prefix for AA, BB and CC respectively. The length of all prefixes are less than 20.

  • n≤30n≤30

Output Format

You should write CC into files in the same file name convention as AA and BB, but with type int64_t.
