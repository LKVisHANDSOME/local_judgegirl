50014. Selection

I'm a slow walker, but I never walk backwards.

Problem Description

Given a set of nn positive integers and another positive integer KK, is there a way to pick a subset of size exactly SS from the set so that the sum of the numbers in the subset is exactly KK? For example, the set is {1,2,3,3,2}{1,2,3,3,2}, KK is 77, SS is 33, then the answer is yes, since we can find a subset 2,2,32,2,3 whose sum of elements is 77. However if SS is 22 then we cannot find any solution.

You need to implement the following function as described above.

1
int subset(int numbers[], int n, int K, int S);

Subtask

  • 10pt. S is 2.
  • 10pt. S is 3.
  • 10pt. n is 4.
  • 60pt. S is between 4 and 7, and n is no more than 10.
  • 10pt. S is between 1 and n, and n is no more than 30.

Hint

To solve the last subtask you need to notice that if the sum of the numbers you selected is already more than KK, you should stop the recursion.
