50166. Newton's Method

I'm a slow walker, but I never walk backwards.

Task Description

Implement Newton's method, which is a root-finding algorithm.
Let ff be a differentiable function defined on the interval (a, b).
To simplify the task we will assume that ff is a polynomial of xx.
We start with x1x1 and find the line at (x1,ƒ(x1))(𝑥1,ƒ(𝑥1)) with slope f′(x1)f′(x1).
This line will intersect with the xx-axis at (x2,0)(x2,0).
We then repeat this procedure to find x3,x4x3,x4, and so on.
Please refer to the following illustration.

Your program will iterate Newton's method for k times and output xk𝑥k and ƒ(xk)ƒ(𝑥k) in each iteration.

Input Format

There are four lines in the input.
The first line is the degree dd of polynomial.

  • 1<d<101<d<10

The second line contains the coefficients of the polynomial and each of them should be double.
For example: 1.54x3+3.88x−4.7421.54𝑥3+3.88𝑥−4.742 is denoted as 1.54 0.0 3.88 -4.742.

The third line is the maximum iteration kk.

  • 0<k<10000<k<1000

The last line is x1𝑥1. It should be double.

Output Format

You need to output xixi and f(xi)f(xi) for every iteration. Please "%.4f" in printf function to output the answers.
