50085. Tank Simulation

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Write a program to simulate a tank movement on a map.

Task Description

We have an NN by MM map with an LL by WW tank and OO obstacles. The vertical index is from 00 to N−1N−1, and the horizontal index is from 00 to M−1M−1. An integer in every cell of the map indicates the status of the cell. A 00 is a space, a 11 is a tank, and a 22 is an obstacle. Note that we will use the upper left corner of the tank to indicate its position, and the initial position of the tank is (0,0)(0,0). The tank will move according to instructions. Each instruction is a number from 00 to 55, indicating the direction the tank should go or we should print the map. Let the current position of the upper left corner of the tank be (x,y)(x,y).

  • If the instruction is 00, output the current map information.
  • If the instruction is 11, the tank moves from (x,y)(x,y) to (x+1,y)(x+1,y).
  • If the instruction is 22, the tank moves from (x,y)(x,y) to (x,y+1)(x,y+1).
  • If the instruction is 33, the tank moves from (x,y)(x,y) to (x−1,y)(x−1,y).
  • If the instruction is 44, the tank moves from (x,y)(x,y) to (x,y−1)(x,y−1).
  • If the instruction is 55, the tank moves from (x,y)(x,y) to (x+1,y+1)(x+1,y+1).

The coordinate system is defined as follows:

p10172. Coordinate

Note that if an instruction moves the tank out of the map or hit more than one obstacle, the tank will not move.
Otherwise, the tank will move in the intended direction.

Write a program to simulate a tank movement on the map and output the map information if the instruction is 00. The program moves the tank, initially located at (0,0)(0,0), according to the input instructions until the end of the file. There is at least one instruction, and the initial area of the tank has no obstacles.

Subtask

  • 20 points: The tank size is 11 by 11 and only have instructions 00~44.
  • 50 points: The tank size is LL by WW and only have instructions 00~44.
  • 30 points: The tank size is LL by WW and have all instructions.

Input Format

The input contains only one test case. The first line contains four integers, NN, MM, LL, and WW representing the length and width of the map and the tank. The second line contains one integer OO representing the number of obstacles. For the following OO lines, each line contains two integers, representing the xx, yy coordinate of obstacles.
The last line contains the instructions for the tank, and the number of instructions is at least one.

  • 0<N,M,L,W<5000<N,M,L,W<500
  • 0≤O<N×M−L×W0≤O<N×M−L×W

Output Format

Print the current map information if the instruction is 00.
