241. Origin in Quadrilateral

I'm a slow walker, but I never walk backwards.

Task Description

Given a convex quadrilateral QQ, please determine if the origin (0,0)(0,0) is with QQ. For example, if QQ has four corners (2,1)(2,1), (−1,2)(−1,2), (−2,−1)(−2,−1), and (−1,−3)(−1,−3) then the answer is yes. If QQ is (12,1)(12,1), (9,2)(9,2), (8,−1)(8,−1), and (9,−3)(9,−3) then the answer is no.

給一個凸四邊形 QQ 的四個頂點座標，請問原點 (0,0)(0,0) 是否在四邊形內部，如果原點在四邊形內部，輸出 1，反之，請輸出 0。

p241-example 1: origin in QQ
p241-example 2: origin not in QQ

Input

The input consists of eight integers, a,b,c,d,e,f,ga,b,c,d,e,f,g, and hh, representing the four corners (a,b),(c,d),(e,f)(a,b),(c,d),(e,f), and (g,h)(g,h) in counterclockwise order. It is guaranteed that the four sides of the QQ will not be parallel to either x or y axis, and will not contain the origin. The absolute value of all numbers is no more than 100.

輸入包含八個整數 a,b,c,d,e,f,g,ha,b,c,d,e,f,g,h，按照逆時針順序給定座標 (a,b),(c,d),(e,f),(g,h)(a,b),(c,d),(e,f),(g,h)。請注意：四邊形的邊不一定平行於座標兩軸，座標的絕對值小於等於 100100。

Output

The output is either 1 or 0, representing that the origin is within or not within QQ.

Sample Input 1

2 1 -1 2 -2 -1 -1 -3

Sample Output 1

1

Sample Input 2

12 1 9 2 8 -1 9 -3

Sample Output 2

0

Origin in QQ if and only if it must satisfy that a⃗ ×b⃗ =2⋅2−1⋅(−1)=5>0a→×b→=2⋅2−1⋅(−1)=5>0, b⃗ ×c⃗ =(−1)⋅(−1)−2⋅(−1)=3>0b→×c→=(−1)⋅(−1)−2⋅(−1)=3>0, c⃗ ×d⃗ =(−2)⋅(−3)−(−1)⋅(−1)=3>0c→×d→=(−2)⋅(−3)−(−1)⋅(−1)=3>0, and d⃗ ×a⃗ =(−1)⋅(1)−(−3)⋅(2)=5>0d→×a→=(−1)⋅(1)−(−3)⋅(2)=5>0
