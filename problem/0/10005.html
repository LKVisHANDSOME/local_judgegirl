Task Description

從一段英數夾雜的字串中，擷取整數輸出。

Input Format

有多組測資，每組測資一行，字串長度不超過 1000，出現的整數 xx 滿足 −231≤x≤231−1−231≤x≤231−1

Output Format

將每一個整數輸出一行。

Sample Input

P0C002NTUCSIE2015Week2EXAMPROBLEM1

Sample Output

0
2
2015
2
1
