10009. Reduce a Fraction

I'm a slow walker, but I never walk backwards.

Task Description

將一個分數化簡到最簡分數。

Input Format

輸入有多組測資，每一組一行兩個整數 a,ba,b。

  • 80 pt. 0≤a≤10000≤a≤1000, ≤b≤1000≤b≤1000
  • 20 pt. 109≤a≤2×109109≤a≤2×109, 109≤b≤2×109109≤b≤2×109

Output Format

對於每組測資輸出一行 x / y。

Sample Input

60 80

Sample Output

3 / 4
