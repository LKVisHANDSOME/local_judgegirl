Task Description

We have two arrays AA and BB of size NN. The elements in AA are distinct positive integers, and the elements in BB are integer pointers. Each pointer in BB will point to an element in AA, and each element in AA will be pointed by exactly one pointer in BB.

sample

We will find loopsloops in AA. For example, in the figure above if we start from ‘1096’ and then follow the pointer in BB, we will visit ‘1096’, ‘894’, ‘214’, then finally ‘1096’. This forms a loop (‘1096’, ‘894’, ‘214’). Since every element in BB points to an element in AA, and every element in AA is pointed by an element in BB, every element in AA is in a loop. In the example above we can find fourfour loops in AA marked by arrows in four different colors. We can find all loops by following the pointers in B, as in the following fgiure.

sample

Write a function loopsloops to determine the the maximum element in the longest loop (and its length), and the minimum element in the shortest loop (and its length). For example in the figure above the the length of the longest loop is 3, and 8600 is the maximum in it. The length of the shortest loop is 1, and 3000 is the minimum in it. Your loopsloops should place these information in the ansans array according to the order defined in the output format section below. Note that there may be multiple longest loops, and you need to find the maximum element in A among them. The same is true for the shortest loops, but you need to find the minimum element among them instead.

The prototype of the loops.hloops.h is as follows.

void loops(int N, int *A, int *B[], int ans[4]);

You can use the following main.cmain.c to test your partition function.

#include <stdio.h>
#include <stdlib.h>
#include "loops.h"
#define MAXN 1000000
int main(){
    int N;
    int *A = (int *)malloc(sizeof(int) * MAXN);
    int **B = (int **)malloc(sizeof(int *) * MAXN);
    scanf("%d", &N);
    for(int i = 0; i < N; i++)
        scanf("%d", A + i);
    for(int i = 0, ptr; i < N; i++){
        scanf("%d", &ptr);
        B[i] = A + ptr;
    }
    int ans[4];
    loops(N, A, B, ans);
    for(int i = 0; i < 4; i++)
        printf("%d\n", ans[i]);
    return 0;
}

Input Format (for the test main program)

The first line contains one integer NN. The second line has NN distinct positive integers. And the third line is a permutation from 0 to N−1N−1, representing the indices of the elements BB needs to point to in AA.

  • 1≤N≤1061≤N≤106

Output Format (for the test main program)

There are four integers in the output (ansans orderly): the length of the longest loop, the length of the shortest loop, the maximum in the longest loop, and the minimum in the shortest loop. You need to place those values in ans in order. Do not print anything in your “loops"“loops".

Hint

If you want to solve the third subtask, use an array visitedvisited of size NN to record whether an element of AA has been visited or not. When you want to find the next loop in AA, you should start from an element in A whose corresponding element in visitedvisited indicates that this element of AA has notnot been visited.
