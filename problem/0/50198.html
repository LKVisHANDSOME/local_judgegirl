50198. Two Digits

I'm a slow walker, but I never walk backwards.

Task Description

Write a program to repeat a computation until it finds a number having at most two different digits. We have three integers aa, bb, and xx. We then repeatedly compute (a×x)modb(a×x)modb and assign the result to xx until xx consists of at most two different digits. We then print xx and stop. Note that if xx consists of no more than two different digits at the beginning, we print it and stop. For example, if xx is/becomes 12121 or 44, we stop. Otherwise, we repeat the process until xx consists of at most two different digits. We guarantee that the computation will stop, and there will be no overflow during the calculation.

Input Format

The input has multiple lines of three positive integers -- aa, bb, and xx. You need to process all lines until EOF.

Output Format

Each output line has the xx when the computation stops.

Sample Input

1234 999 112
1 99 2345667
10000 5678 1234

Sample Output

112
60
550

Estimated Cyclomatic Number

7.42
