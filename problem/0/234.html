Task Description

Write a program to factorize a polynomial. We will be given a polynomial f(x)=x3+ax2+bx+cf(x)=x3+ax2+bx+c, where all a,b,ca,b,c are all integers between -100000 and 100000. Now we want to factorize f(x)f(x) as (x+d)(x+e)(x+f)(x+d)(x+e)(x+f), where dd, ee, and ff are all integers, and d≤e≤fd≤e≤f.

Input

A line containing aa, bb, and cc.

Output

A line containing dd, ee, and ff.

Sample input

12 47 60

Sample output

3 4 5
