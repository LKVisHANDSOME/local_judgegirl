Task Description

Given NN binary files and each file contains multiple four-byte integers in ascending order, write a program to read the files and print all integers in those files in ascending order. For example, If we are given three files 1.dat1.dat, 2.dat2.dat, and 3.dat3.dat. The contents of the files are as follows (in decimal). 

| 1.dat | 2.dat | 3.dat | 
|-------|-------|-------|
|  5    |  2    |  19   |
|  14   |  23   |  55   |
|  27   |  36   |       |
|       |  40   |       |

We should output the integers in the following order.

2
5
14
19
23
27
36
40
55

Hint

You should first read one integer from all files, so you have NN integers. Then you repeat the following process. Print the smallest one among these NN integers. If that smallest integer is from file FF, then replace the smallest integer with the next integer from file FF. You repeat this process until you have read all integers from all files.

Input Format

There is one test case. The first line contains an integer NN indicating the number of binary files.
Each of the following NN lines contains a string SS indicating the name of a file.
0<N<16.0<N<16.
0<0< length of S<16.S<16.

Output Format

Print all integers on separate lines.
