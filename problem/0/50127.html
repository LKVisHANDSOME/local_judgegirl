50127. Connect the Numbers

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Task Description

We have a two-dimensional NN by NN array AA where every cell has a distinct integer from 00 to N2–1N2–1. Also we have a two-dimensional NN by NN array BB where each cell has a pointer to an integer. Now we need to connect the numbers of array AA with the pointers in BB. That is, if a cell of AA has the integer ii, then its corresponding cell in BB should point to a cell in AA that as i+1i+1. To make the linkage well defined, if a cell of AA has the integer N2−1N2−1, then its corresponding cell BB should point to a cell in BB that as 00.

We illustrate this linkage by the following figure. A[2][0]A[2][0] is 00 and A[1][2]A[1][2] is 11, so B[2][0]B[2][0] should point to A[1][2]A[1][2]. Now A[0][0]A[0][0] is 22, so B[1][2]B[1][2] should pointe to A[0][0]A[0][0], and so on. Finally, Since A[2][2]A[2][2] is 88 and A[2][0]A[2][0] is 00, B[2][2]B[2][2] should point to A[2][0]A[2][0].

figure

You need to write a function constructPointerArray, that given array AA and BB, fill the values of element in BB. The prototype of constructPointerArray function is as follows. Note that you cannot chanage the elements in A. You can change the elements in B, but not the value they point to.

void constructPointerArray(const int A[1024][1024], const int *B[1024][1024], int N);

The constructPointerArray.h is as follow.

#ifndef CONSTRUCT
#define CONSTRUCT
#define MAXN 1024
void constructPointerArray(const int A[MAXN][MAXN], const int *B[MAXN][MAXN]
, int N);
#endif

You may use the following main function to test your function.

#include <stdio.h>
#include "constructPointerArray.h"
int A[MAXN][MAXN];
const int *B[MAXN][MAXN];
int main()
{
    int N;
    scanf("%d", &N);
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < N; j++) {
            scanf("%d", &A[i][j]);
        }
    }
    constructPointerArray(A, B, N);
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < N; j++) {
            printf("%d%c", *B[i][j], j == N - 1 ? '\n' : ' ');
        }
    }
    return 0;
}

Input Format

The input contains only one test case. The first line contains one integer NN, representing the size of array AA. Each of the following NN lines contains NN integers, representing the elements of array AA.

  • N≤1024N≤1024

Output Format

Print the number pointed to by pointer array BB. Note that we will also check the value of B to be within the address of A, no only the values pointed by B.

Hint

For subtask 2 it would be time consuming to find the location of a particular number i+1i+1 within AA. You can use another array DD where D[i]D[i] stores the address of ii in A. Then we can fill the array DD after scanning AA, and fill the BB by pass through DD.
