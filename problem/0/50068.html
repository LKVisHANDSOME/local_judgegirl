50068. Tree Traversal

I'm a slow walker, but I never walk backwards.

Task Description

Given the root of a binary tree and a series of N commands, write a function traversal to traverse the tree and output the result.

p134
The node is given by a structure Node.

1
2
3
4
typedef struct Node {
    int label;
    struct Node *left, *right;
} Node;

There are six types of commands, and the command type tt is an integer from 00 to 55. Let the current node of the tree be root:

  • t=0t=0 - stop traversing and print the label of the current node.
  • t=1t=1 - print the label of the current node.
  • t=2t=2 - go to the parent of the current node.
  • t=3t=3 - go to the left child of the current node.
  • t=4t=4 - go to the right child of the current node.
  • t=5t=5 - go to the sibling of the current node. The sibling is the other node that have the same parent as you do. For example, in the picture above node 11 and 55 are siblings.

A command is invalid if the destination node specified does not exist. An invalid command will stop the traversal and your program should print the label of the current node and ignore the rest of the commands.

The prototype of the function you need to implement is as follows:

1
void traversal(Node *root, int N, int command[]);

Hint

You can use an array to store the address of nodes along the path from the root to the current node. Then all the operations are very easy.

Input Format

The input tree will be given as a pointer to the root and NN commands.

  • 0<N<10000<N<1000

Output Format

Print the label of current node if the command is 11 and the label of the last node before you stop the traversal on separate lines.
