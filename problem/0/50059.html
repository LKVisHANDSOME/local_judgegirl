50059. Binary Representation

Task Description

Given NN 32-bit non-negative integers and a string SS consists of 3232 characters, represent the integers using characters in SS.

A 32-bit integer has 32 bits, so after we concatenate NN of them, we will get a bit string of length 32⋅N32⋅N. For example, if we have two integers 10949755401094975540 and 5252, then the binary representation of 10949755401094975540 is 01000001010001000000000000110100, and that of 5252 is 00000000000000000000000000110100. After concatenating them we will have 0100000101000100000000000011010000000000000000000000000000110100.

Your task is to form every five bits into a group, starting from the most significant bit and ignore the trailing bits (忽略最後不滿五個的 bits). We will have ⌊N⋅325⌋⌊N⋅325⌋ groups for a bit string of length N⋅32N⋅32. Each group contains five bits, which is a binary representation of an integer in range [0,25)[0,25). For example, the bit string 0100000101000100000000000011010000000000000000000000000000110100 is splitted into 1212 groups as follows:

01000 00101 00010 00000 00000 01101 00000 00000 00000 00000 00000 00011 (0100)

Those groups represent 1212 integers 88, 55, 22, 00, 00, 1313, 00, 00, 00, 00, 00 and 33 respectively.

We use those integers as indices and output the corresponding characters in the given string SS. The index goes from 00 to 3131. For example, given SS = AaBbCcDdEeFfGgHhIiJjKLMNOPQRSTUV, the group mentioned above should be represented by EcBAAgAAAAAb, which are the 88-th, 55-th, 22-nd, 00-th, 00-th, 1313-th, 00-th, 00-th, 00-th, 00-th, 00-th and 33-rd character in SS respectively.

Input Format

Input contains one test case with three lines. The first line contains a string SS with exactly 3232 non-space characters. The second line contains an integer NN, representing the number of 32-bit integers. The third line contains NN non-negative integers.

  • 1≤N≤200001≤N≤20000

Output Format

Output one line with ⌊N⋅325⌋⌊N⋅325⌋ characters, representing the decoded string.

Sample Input

AaBbCcDdEeFfGgHhIiJjKLMNOPQRSTUV
2
1094975540 52

Sample Output

EcBAAgAAAAAb
