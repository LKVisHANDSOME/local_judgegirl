190. Function Evaluation

I'm a slow walker, but I never walk backwards.

Write a function to evaluate f(x,y)=4∗x−6∗yf(x,y)=4∗x−6∗y for a lot of xx and yy. The prototype of the function is as follows.

1
int evaluate_f(int *iptr[], int n, int *index);

Each pointer in iptr points to an array of two integers. The first integer is xx, and the second integer is yy. The length of the pointer array is nn, so there are nn pairs of xx and yy that you need to evaluate f(x,y)f(x,y). Compute these f(x,y)f(x,y) and return the maximum as the return value of evaluate_f. Also you need to set index so that it becomes the index into iptr where the maximum happens. If the maximum value can be evaluated from multiple (x,y)(x,y) pairs, set the index to be the smallest among them. For example, if the xx and yy pointed by iptr[3] and iptr[5] both have the maximum f(x,y)=100f(x,y)=100, then evaluate_f should return 100100 and index should be set to 33. Variable nn is always positive.

n > 0
