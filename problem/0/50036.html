50036. Pick up Numbers

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Task Description

Write a program to count the number of ways to pick at least kk number out of NN positive numbers so that the sum of the picked numbers is no more than MM.

For example, given four numbers 11, 33, 55, and 77, then there are 66 ways to pick at least 22 numbers so that their sum is no more than 1010. The solutions are {1,3},{1,5},{1,7},{3,5},{3,7}{1,3},{1,5},{1,7},{3,5},{3,7}, and {1,3,5}{1,3,5}.

Input Format

The first line contains three integers: kk, NN and MM, where 1≤k≤N1≤k≤N, 1≤N≤201≤N≤20, and 1≤M≤50000001≤M≤5000000. The second line contains NN positive integers; each of them is smaller than 100000100000.

Output Format

The number of ways to pick at least kk number out of NN positive numbers so that the sum of the picked numbers is no more than MM.

Sample Input

2 4 10
1 3 5 7

Sample Output

6
