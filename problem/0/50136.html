50136. Build Strings

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics

Task Description

Write a program to generate strings according to three constraints. First the first letter of this string must be a given letter CC. Next the length of the string must be a given LL. Finally, the distance between a letter and the letter after it must be no more than KK. For example, if CC is 'a' and KK is 4, then the second letter of this string can only be one from the set {'b', 'c', 'd', 'e'}. Note that the distance between the letters is circular. That is if CC is 'w' and KK is 5, the second letter be one from {'x', 'y', 'z', 'a', 'b'} in a wrapped around manner. The same rule applied to every letter in the string, e.g., between the second and the third letter etc.

Write a program to generate strings recursively, and print the first NN strings in dictionary order. All letters are in lowercase.

Input Format

The input contains only one test case. There are one character CC and three integers KK, LL, and NN in one line. CC is the leading character of strings, KK is the maximum distance between consecutive letters, LL is the length of the strings, and NN is the number of strings you need to print.

  • 0<K<260<K<26
  • 0<L<10240<L<1024
  • 0<N≤KL−10<N≤KL−1

Output Format

Output contains NN lines. There is a string in each line.

Subtasks

  • 10 points: KK is 1, and there will be no wrapped around manner. NN is so large that you need to print all possible strings.
  • 30 points: There will be no wrapped around manner. NN is so large that you need to print all possible strings.
  • 30 points: There will be no wrapped around manner, but you need to print only the first NN strings.
  • 30 points: There could be wrapped around manner, and you need to print only the first NN strings.
