50042. Highest Mountain

I'm a slow walker, but I never walk backwards.

Task Description

Given a matrix of mountains, write a program to find the positions of the highest and the second highest mountains for every row, and the position of the overall highest mountain.

There is a matrix of mountains of nn rows and mm columns. Let MijMij denote the mountain in the ii-th row and jj-th column. Also let hijhij be the height of mountain MijMij. The row index goes from 11 to nn, and the column index goes from 11 to mm.

We want to report the column index of the highest and the second highest mountains in a row. If there are multiple highest mountains in the same row, we report the one with the largest column index as the highest mountain, and the one with the second largest column index as the second highest mountain. If there are multiple second highest mountains, we report the one with the largest column index.

We also want to report the row and column index of the overall highest mountain. If there are multiple highest mountains, then report the one with the largest row index. If there are still multiple highest mountains in that row, report the one with the largest column index.

A 33 by 44 example here.

⎡⎣⎢200103312403⎤⎦⎥[213400100323]

For the ii-th row we define pcipci as the column index of the highest mountain and qciqci as the column index of the second highest mountain. For all rows we define (ax,ay)(ax,ay) as the (row index, column index) of the highest mountain.

For example, the first row is [2,1,3,4][2,1,3,4], the second row is [0,0,1,0][0,0,1,0] and the third row is [0,3,2,3][0,3,2,3]. In the first row, the highest mountain locate fourth column and the second highest mountain locate third column, so we get (pc1,qc1)=(4,3)(pc1,qc1)=(4,3). Similarly, according to the above mentioned definition, we get (pc2,qc2)=(3,4)(pc2,qc2)=(3,4) and (pc3,qc3)=(4,2)(pc3,qc3)=(4,2). In all rows, the highest mountain locate the first row and the fourth column, as a result, we get (ax,ay)=(1,4)(ax,ay)=(1,4).

Now given the heights of all mountains, report all positions described above.

Input Format

There is one test case.

The first line of the input file contains two integer nn, mm, representing the number of rows and columns. Each of the next nn lines contains mm integers, representing a matrix of the mountains, where the jj-th integer on line ii denotes the height hijhij of the mountain MijMij.

Output Format

For each test case, the output contains n+1n+1 lines. Each of the next nn lines of output consist of two integers pc,qcpc,qc separated by a single space, depending on the column index of the highest and second highest of mountain in the ii-th row. The last line of output consists two integers ax,ayax,ay by separated by a single space, depending on the row and column index of the highest of mountain in all rows.
