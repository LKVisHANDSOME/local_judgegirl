50063. Friend Distance

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Task Description

We have NN people and MM pair of friends among them, and we would like to assign people to a line of NN chairs to minimize the maximum distance between two friends.

We illustrate the seating with examples. We use the first NN uppercase letters to indicate people. Let NN be 8, and we seat people as ABCDEFGH, with A and F are friends, then the distance between them is 5. Furthermore, if A and H are also friends, then the maximum distance of this ordering is 7.

Subtask

  • 95pt. 1<N≤101<N≤10.

  • 5pt. N=11N=11. You need to cut the search for this subtask once you realize that any further selection from this variation will not be better than the best one you have seen. If you know the current best solution is six and your current selection already has a distance of eight, then it is meaningless to continue.

Input Format

There is only one test case. The first line contains two integers indicated NN and MM; then, each of the following MM lines has two uppercase letters for friends.

Output Format

The first line contains minimized maximum distance between two friends. The second line is the first seating order that minimizes the distance in lexicographical order.

Sample Input 1

3 1
A B

Sample Output 1

1
A B C

A F
B C

Sample Output 2
