230. The Knapsack

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Task Description

假設有 nn 個物件和一個背包。第 ii 個物件的重量為 wiwi，價值為 vivi，而背包有總重量限制 WW。如何選擇物件放入背包中，使得放入背包的物件總重量不超過背包總重量量限制 WW，且使得放入背包的物件總價值為最大？

Input

輸入第一行為物件個數 nn 及背包重量限制 WW。接下來會有 nn 行，每行有兩個數字，分別是第i個物件的重量 wiwi 和價值 vivi。

Limits

  • 0<n≤200<n≤20
  • 0<W≤100000<W≤10000
  • 0<wi,vi≤10000<wi,vi≤1000

Sample input　

5 6
3 1
1 3
2 3
3 5
3 5

Sample output

11

HINT

可以使用遞迴的方法來求答案。如果我們將第一個物件放入背包，結果是不是得到一個跟原來問題很相近的問題? 如果我們不將第一個物件放入背包，結果是不是也得到一個跟原來問題很像的問題？
