50119. Paper, Scissors, Stone

I'm a slow walker, but I never walk backwards.

Task Description

You are given two integer sequences X(t)X(t) and Y(t)Y(t) for tt = 0, 1, 2, .... Sequence X(t)X(t) is generated by the formula X(t+1)=(aX(t)+b)(modc)X(t+1)=(aX(t)+b)(modc). And sequence Y(t)Y(t) is generated by the formula Y(t+1)=(dY(t)+e)(modf)Y(t+1)=(dY(t)+e)(modf). aa, bb, cc, dd, ee, and ff are positive integers.

Write a program to play paperpaper, scissorsscissors, stonestone with the two integer sequences. For any integer XX, we can determine whether XX is paperpaper, scissorsscissors, or stonestone with the value of Xmod3Xmod3. XX is paperpaper when the value of Xmod3Xmod3 is 0. XX is scissorsscissors when the value of Xmod3Xmod3 is 1. And XX is stonestone when the value of Xmod3Xmod3 is 2. ScissorsScissors beat paperpaper, stonestone beats scissorsscissors, and paperpaper beats stonestone.

We will have many rounds of games by two players. In a round of game, we pick a new integer pair (X(t)(X(t), Y(t))Y(t)) from each sequence and check which player wins. We keep picking new integer pairs until one player wins this round.

For example, X(0)X(0) is 4 , and X(t+1)=(5X(t)+6)mod7X(t+1)=(5X(t)+6)mod7. Also Y(0)Y(0) is 6, and Y(t+1)=(5Y(t)+4)mod3Y(t+1)=(5Y(t)+4)mod3. We play three rounds. In the first round, we pick (X(0)(X(0), Y(0))=(4Y(0))=(4, 6)6). X(0)X(0) is scissorsscissors, Y(0)Y(0) is paperpaper, so XX wins this round. In the second round, we have (X(1)(X(1), Y(1))=(5Y(1))=(5, 1)1). X(1)X(1) is stonestone. Y(1)Y(1) is scissorsscissors, so XX wins this round. In the third round, we have (X(2),Y(2))=(3,0)(X(2),Y(2))=(3,0). Both X(2)X(2) and Y(2)Y(2) are paperspapers, so this pick ends in a draw. We keep picking a new pair (X(3),Y(3))=(0,1)(X(3),Y(3))=(0,1). X(3)X(3) is paperpaper, Y(3)Y(3) is scissorsscissors, so YY wins this round.

Input Format

The input contains only one test case. There are four positive integers X(0)X(0), aa, bb, and cc in the first line. There are four positive integers Y(0)Y(0), dd, ee, and ff in the second line. And there is one positive integer NN in the third line. NN is number of round we will play.

Output Format

The output contains NN lines. There are two integers ww and kk in each line. The first integer ww determines which player wins this round. When XX wins, ww is 0, and ww is 1 when YY wins. The second integer kk determines how many pairs we picked in this round.

Subtask

  • 10 points: NN is 1. We play only one round, and the are no draws in the game.
  • 10 points: NN is larger than 1, and there are no draws
  • 80 points: NN is larger than 1, and there are draws in the games.
