50193. Minimum Time Difference

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Task Description

Given a sequence of digital timestamps, find the minimum time difference between two consecutive timestamps in the sequence. A timestamp has 6 digits. The first two digits are hours, the second two digits are minutes, and the last two digits are seconds. For example, 152623 means 15:26:23.

Suppose we have the sequence is 152623, 182459, and 125641. The time difference between 15:26:23 and 18:24:59 is 2:58:36, while the time difference between 18:24:59 and 12:56:41 is 5:28:18. Since 2:58:36 is less than 5:28:18, the minimum time difference between two consecutive timestamps is 2:58:36.

Input Format

The input is a sequence of at least two valid timestamps, each in its line. You need to process all timestamps until EOF.

Output Format

The output has the minimum time difference between two consecutive timestamps in the input sequence. The output format is in seconds.

Sample Input 1

152623
182459
125641

Sample Output 1

10716

Sample Input 2

001526
231124
052632
140503
010045
221200
144535

Sample Output 2

26785
