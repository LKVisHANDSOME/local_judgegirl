50128. Split a List

I'm a slow walker, but I never walk backwards.

Task Description

We have two arrays AA and aa. The elements of AA are positive integers, and the elements of aa are pointers to integer. We initialize a so that a[i]a[i] pointing to the A[i+1]A[i+1], and the last element of aa is NULL. This is like a list of integers. Please refer to the following figure.

sample

Now we want to split this list of integers into kk lists. We have a pointer array of kk elements (headhead) as the the heads of these kk lists. The list of AA is split into kk lists according to the remainder when divided by kk. That is, all integers of AA in the form as {kn}{kn} will be in the first list, and those as {kn+1}{kn+1} will be in the second list, and so on. Please refer to the following figure for an illustration. For example, we have k=5k=5 and the first list (starting from head[0]head[0]) will have 3535 (in A[1]A[1]) and 6565 (in A[4]A[4]). Note that there is no element after 6565 so a[4]a[4] will be NULL. Also note that if there are no elements in a list, the corresponding headhead will be NULL. For example, no element in AA is 5n+35n+3, so head[3]head[3] is NULL.

sample
Write a function split to split the elements in AA into kk lists as described above.

The prototype of split.h is as follows. (Remember to include stdlib.h in split.c.)

void split(int A[], int *a[], int *head[], int k);

You can use the following main.c to test your function.

#include <stdio.h>
#include <stdlib.h>
#include "split.h"
 
int main(int argc, char const *argv[])
{
    int N, k;
    scanf("%d%d", &N, &k);
    int A[N];
    for (int i = 0; i < N; ++i)
        scanf("%d", &A[i]);
    int *a[N], *head[k];
    for (int i = 0; i < N-1; ++i)
        a[i] = &A[i+1];
    a[N-1] = NULL;
    for (int i = 0; i < k; ++i)
        head[i] = NULL;
    split(A, a, head, k);
    for (int i = 0; i < k; ++i) {
        if (head[i] == NULL)
            printf("NULL\n");
        else {
            int *tmp = head[i];
            printf("%d", *tmp);
            tmp = *(tmp-A+a);
            while (tmp != NULL) {
                printf(" %d", *tmp);
                tmp = *(tmp-A+a);
            }
            printf("\n");
        }
    }
    return 0;
}

Input Format (for the test main program)

We will give you AA, aa, headhead, and kk as the input of function split. AA and aa have the same size NN. The size of headhead is kk. For main.c, the input will be NN, kk in the first line, and elements of array AA in the second line.

  • 1≤N<1000001≤N<100000
  • 1≤k<1000001≤k<100000

Output Format (for the test main program)

For main.c, the output has kk lines representing kk lists from {kn}{kn} to {kn+(k−1)}{kn+(k−1)}. Each line shows elements in one list according to the pointing order. If there is no element in a list, that line will output NULL.

Hint

It is more convenient to use an array tailtail to record the current last integer of each list.
