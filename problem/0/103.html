103. Triangular Matrix Solver

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Task Description

Write a program to solve a system of equations. In particular we are given an nn by nn upper triangular matrix AA and a nn by 11 vector yy, and we would like to find another nn by 11 vector xx so that Ax=yAx=y. Since AA is upper triangular, i.e., all the elements below the diagonal are zero, we can use a simple procedure called backward substitution to get the vector xx. Since An,n×xn=ynAn,n×xn=yn, so we conclude that xn=yn/An,nxn=yn/An,n. Since we know xnxn now, we can easily compute xn−1xn−1, then xn−2xn−2, and so on, until we finally compute x1x1.

Input

The first line of the input has the number of rows and columns nn. nn is between 1 and 16. Each of the following nn lines has nn double numbers in AA. Each of the following nn lines has the numbers in yy. Note that all elements of matrix and vector are double numbers.

Output

The output has nn lines. Each line is a number in xx. You should output the double numbers in %f\n format.

Sample input

3
1.0 2.0 3.0
0.0 2.0 1.0
0.0 0.0 4.0
2.0
3.0
-4.0

Sample output

1.000000
2.000000
-1.000000
