50156. Highest Discount Rate

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Task Description

There is a sales coming. Every payment pp (a positive integer) will have a discount dd according to the following table.

range of pp discount dd
1 0<p<a0<p<a 0
2 a≤p<ba≤p<b v, if p is oddv, if p is odd
w, if p is evenw, if p is even
3 b≤pb≤p x, if p≡0(mod3)x, if p≡0(mod3)
y, if p≡1(mod3)y, if p≡1(mod3)
z, if p≡2(mod3)z, if p≡2(mod3)

We define discount rate as d/pd/p. Write a program to find the pp that has the highest discount rate. If there are multiple payments that give the highest discount rate, output the smallest one.

Input Format

There will be 7 lines for integers a,b,v,w,x,y,za,b,v,w,x,y,z.

  • v,w,x,y,z≥0v,w,x,y,z≥0
  • a>1a>1
  • b>a+2b>a+2

Output Format

The smallest payment pp that has the highest d/pd/p discount rate.

Sample Input

10
30
5
6
18
8
9

Sample Output

10

Hint

Since we have not yet discuss the floating point numbers, so we compare two fractional numbers a/ba/b and c/dc/d, by comparing a∗da∗d and b∗cb∗c instead, when both bb and dd are positive integers.
