50051. Valid License Plates

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Task Description

Given NN license plates, output the valid license plates in ASCII order.

There are two kinds of license plates. The first kind has the form x1x2x1x2-x3x4x5x6x3x4x5x6 and the second kind has the form x1x2x3x1x2x3-x4x5x6x4x5x6, where x1x1 ~ x6x6 are non-space characters.

A license plate is valid if it satisfies all the following constraints:

  1. x1x1 ~ x6x6 are all alphabets or digits.
  2. x1x1 ~ x6x6 should contain at least one digit, and the sum of all the digits can not be divided by 77.
  3. x1x1 ~ x6x6 should not contain the same character (both digits and alphabets) more than twice. The alphabets are case sensitive (字母大小寫不同，算不同的字).
  4. x1x1 ~ x6x6 should not contain two or more consecutive digit 44. For example, "ab-44cd" is invalid. Note that x2x2 and x3x3 are not consecutive in the first kind of the license plate, and x3x3 and x4x4 are not consecutive in the second kind of the license plate.

Output the first kind of the valid license plates in ASCII order, i.e '0' < '1' < ... < '9' < 'A' < ... < 'Z' < 'a' < …< 'z'. Then, output the second kind of the valid license plates in ASCII order.

Input Format

Input contains one test case. The first line is an integer NN, indicating the number of the license plates. For the following NN lines, each line contains a string with exactly 77 non-space characters.

Technical limitation

  • 1≤N≤301≤N≤30

Output Format

Output the valid license plates. Output the first kind of the valid license plates in ASCII order. Then output the second kind of the valid license plates in ASCII order.
