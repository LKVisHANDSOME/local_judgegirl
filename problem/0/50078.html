50078. Parallelogram

I'm a slow walker, but I never walk backwards.

Task Description

You are given three points AA, BB, and CC. Please find a point DD so that ABDCABDC becomes a parallelogram. Then find a point EE so that ABCEABCE becomes a parallelogram. Finally, find a point FF so that ACBFACBF becomes parallelogram. Report the coordinates of point DD, then EE, then FF.

sample

Input Format

There are 6 integers in different line, indicating the xx, yy coordinates of AA, BB, and CC.

Output Format

There are 6 integers in different line, indicating the xx, yy coordinate of DD, EE, and FF.
