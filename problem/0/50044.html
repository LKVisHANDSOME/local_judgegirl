50044. Caterpillar in Panic

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Write a program to simulate a caterpillar moving on a grid.

Task Description

Read problems statements in 中文

We have an N×MN×M grid GG of squares with a caterpillar in it. We use an NN by MM array to indicate the positions of the caterpillar. The row index is from 00 to N−1N−1, and the column index is from 00 to M−1M−1. If an array cell is 11 then it is a part of the caterpillar, and if the the cell is 00 then it is a space. A caterpillar is a sequence of LL adjacent 1's on the grid, where two cells are adjacent if either their row or column indices differ by exactly 11. A caterpillar has head (at the starting position of the adjacent 1's), and a tail (at the ending position of the adjacent 1's).

The caterpillar moves on the grid as follows. The caterpillar can either move east, west, south, or north, if the corresponding cell in that direction is a space, or the tail of the caterpillar. Note that a space is within the grid.

The caterpillar moves according to a sequence of directions, where DiDi is the ii-th direction. Assume that the head is located at (x,y)(x,y), then the caterpillar moves as follows.

  • If DiDi equals 0, the head of the caterpillar moves to (x+1,y)(x+1,y).
  • If DiDi equals 1, the head of the caterpillar moves to (x−1,y)(x−1,y).
  • If DiDi equals 2, the head of the caterpillar moves to (x,y+1)(x,y+1).
  • If DiDi equals 3, the head of the caterpillar moves to (x,y−1)(x,y−1).

Write a program to simulate a caterpillar movement on the grid. The program simulates the movement according to directions or until it violates the rules of movement. Finally, output the values of the N×MN×M grid.

Hint

You can use three arrays to store the current status of the caterpillar. The first NN by MM array stores the status of grid GG, and another two arrays of length LL for the current row and column indices of each part of the caterpillar.

Input Format

There is one test case.

The first line contains two integers N,MN,M, representing the height and width of grids. The second line contains four integers SX,SY,EX,EYSX,SY,EX,EY, representing the position of head and tail. The test set guarantee the snake parallel one axis. The third line contains an integer QQ, representing the the number of directions. The following line contains QQ integers DD, representing the direction of movement each time.

  • 0<N,M≤10000<N,M≤1000
  • 0≤SX,EX<N0≤SX,EX<N
  • 0≤SY,EY<M0≤SY,EY<M
  • 0≤Q≤10000000≤Q≤1000000
  • D∈{0,1,2,3}D∈{0,1,2,3}

Output Format

Output file contains NN lines. Each line contains MM 0/1 integers. If the square belongs the caterpillar parts, output 1. Otherwise, output 0.
