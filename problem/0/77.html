77. Get the i-th Element in a List

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Now you have a linked list, write a function to get the ii-th element in this list. The function has the following prototype:

struct node *getNode(struct node *head, unsigned int i);

You only need to implement this function. This function will return a pointer that points to the ii-th element in the linked list, with i=0i=0 being the last element and the indices are counted in reverse order. If ii is too large for the list, you should return NULL. The list won't be empty, and the last node's pointer next will point to NULL.

Please include this line #include "node.h" when submitting.
