50094. Word Learning

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics
  • Solution

Write a program to train words and generate a new word.

Task Description

Given a set of words and a threshold TT, count the number of times a letter follows another letter, and use this information to build a new word. For example, in a word cat c is followed by a and a is followed by t. After processing all words then we will have a 2626 by 2626 table in which each cell represents the number of times a letter followed by another letter.

Now we generate a new word. First we starts with the letter pair that appears the most number of times from the the words we learned. Then we repeated concatenate a letter the appears the most number of times to the end of the current word. Note that we can only use the letter that has not appeared before, and the number of occurrences is larger than the given threshold TT. If there are more than one such letter, choose the one that appears first alphabetically. We repeat the second step until we cannot any letter to continue.

Subtask

  • 3030 points: All lengths of training words are two.
  • 7070 points: Training words may have different lengths.

Input Format

It is guaranteed that all words consist of lowercase letters and all letters of new word are different. The input contains one test case. The first line contains one integer TT, representing the threshold. The rest lines contain words to be learned.

  • 0≤0≤ TT ≤500≤500
  • 2≤2≤ length of training word ≤100≤100
  • 2≤2≤ length of result new word ≤26≤26

Output Format

Print the training table and the new word according to the training.

Note that there is no space at the end of line.
