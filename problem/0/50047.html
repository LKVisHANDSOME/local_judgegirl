50047. Game of Cell

I'm a slow walker, but I never walk backwards.

Task Description

Write a function game_of_cell to determine the outcome of a Game of Cell on an 5050 by 5050 grid after NN rounds.

Game of Cell is a game on an 5050 by 5050 grid of square cells. Each cell is either healthy, dying or dead. The future status of each cell is determined by the status of its current neighbors, which are up, down, left, right, or diagonally adjacent to it.

  1. A healthy cell with fewer than two healthy neighbors will become dead.
  2. A healthy cell with two or three healthy neighbors will not change status.
  3. A healthy cell with more than three healthy neighbors will become dying.
  4. A dead cell with exactly three healthy neighbors will become healthy. Otherwise it remains dead.
  5. A dying cell with exactly two healthy neighbors will become healthy. Otherwise it remains dying.

The prototype of game_of_cell is as follows.

1
void game_of_cell(int grid[50][50],int outcome[50][50],int N);

The grid[50][50] is the grid, indicating the initial status of each cell. A 0 indicates a dead cell, a 1 indicates a dying cell, and 2 indicates a healthy cell. After NN rounds, the status should be placed into the array outcome[50][50]. You only need to write the function and the header, and we will use a main program to test the correctness of your function.

    scanf("%d",&N);
    for(int i=0;i<50;i++)
        for(int j=0;j<50;j++)
            scanf("%d",&grid[i][j]);

        for(int j=0;j<50;j++)
            printf("%d%c",outcome[i][j]," \n"[j==49]);
    return 0;
}

Output Format

There are 50 lines,each of which has 50 elements in the output.The 50 lines are the outcome grid after NN rounds of the game.
