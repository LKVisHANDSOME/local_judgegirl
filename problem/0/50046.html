50046. Linear Classifier

I'm a slow walker, but I never walk backwards.

  • Status
  • Statistics

Write a simple machine learning program. In the first training stage, we train a model. In the second prediction stage we classify inputs according to the model we trained.

Task Description

Read problems statements in Mandarin Chinese

An image classifier classifies input images into two categories -- accept and reject. An Image is an MM by MM array PP of pixels, where each pixel pi,jpi,j is either zero or one. A classifier determines which category an image belongs to based on a function value h of all pixels of this image. There is a weight wi,jwi,j for every pixel pi,jpi,j, and the hh function value of an image is the sum of products of all pixel values with their corresponding weights.(h(P)=∑i,jpi,jwi,jh(P)=∑i,jpi,jwi,j) If the hh value is no less than a threshold TT, which is 2M22M2, then the model accepts the image, otherwise it rejects the image.

The classifier starts with an initial weights of all 1's, then modify them according to a set of NN train inputs. Each input is an MM by MM array of 0 or 1's, and has a label of either accept or reject. If the model from the current set of weights gives the same result as the label of the input image, then we do nothing. Otherwise we have two cases. In the first case the classifier accepts the input but the label is reject. Then we divide the weight of those non-zero pixel by 22, and set the weight to 11 if the the division gives 00. In the second case the classifier rejects the input but the label is accept. Then we multiply the weight of those non-zero pixel by 2.

After we train our model we use it to classify another set of QQ test inputs. We evaluate the hh function using the weights we obtained in the previous stage, and accept the input if the hh value is no less than the threshold, and reject the input otherwise.

Input Format

There is one test case. Each test case has two parts.

In the first part, the training data set contains NN images. The first line contains two integers N,MN,M, representing that there are NN images which size is M×MM×M as follows: Each image contains M+1M+1 lines. In M+1M+1 lines, the first line contains an integer CC, representing the result of each images. In following MM lines, each line contains MM integers.

In the second part, the query data set contains QQ images. The first line contains an integer QQ, representing QQ images as follows: Each image contains MM lines. Each line contains MM integers.

  • 2≤N≤20,1≤M≤162≤N≤20,1≤M≤16
  • 0≤Q≤10000≤Q≤1000

Output Format

For each query, output one line, which contains an 0/1 integer.
